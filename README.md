# Zenn CLI

- [📘 How to use](https://zenn.dev/zenn/articles/zenn-cli-guide)

### 新しい記事を投稿する
**記事の作成**
```
    # 記事の作成
    pnpm exec zenn new:article --slug <article-name>-tf63-<20XX0X0X>
    # 記事のpreview
    pnpm exec zenn preview
```

**記事の公開**

マークダウンファイルでpublishedオプションをtrueにしたうえでgithubにpush
